# Progreso del proyecto

Esta carpeta mantiene el estado del desarrollo de Guaranda Go entre sesiones.

## Archivos

- `phases.md`: fases y avance general.
- `current_status.md`: estado actual para retomar trabajo.
- `session_log.md`: bitácora resumida por sesión.
- `decisions_log.md`: decisiones técnicas/producto tomadas durante el desarrollo.

Los agentes deben actualizar estos archivos cuando cambie el estado del proyecto.
