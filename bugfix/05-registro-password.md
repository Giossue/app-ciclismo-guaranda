# 05 — Registro y contraseña

## Problema

El usuario conocía problemas de contraseña solo después de enviar el formulario.

## Cambio aplicado

- Se agregó checklist visual antes de enviar.
- Se valida longitud mínima, mayúsculas/minúsculas, número, símbolo y confirmación.
- El botón se bloquea hasta cumplir los requisitos visibles.
- Se aclara que la revisión contra contraseñas filtradas se confirma en servidor al crear cuenta.
